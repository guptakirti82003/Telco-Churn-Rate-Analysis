# Customer Churn Analysis & Retention Insights (Telco Dataset)

## Problem Statement
Customer churn is a key challenge for telecom companies. This project analyzes customer data to identify churn patterns and provide actionable insights to improve retention.

## Dataset
Telco customer churn dataset containing customer demographics, services, contract details, tenure, and billing information.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Tableau

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Churn analysis by contract type, tenure, and monthly charges
- Aggregations and trend analysis

## Key Insights
- Customers on month-to-month contracts had higher churn
- Higher monthly charges correlated with increased churn
- New customers with low tenure were more likely to churn

## Business Recommendations
- Promote long-term contracts through targeted offers
- Improve onboarding experience for new customers
- Identify high-risk customers early and apply retention strategies
